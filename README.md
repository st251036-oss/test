## 2年3組44番 深山 友希 (Yuki Miyama)

趣味の紹介を通じて、私のことを知っていただければ嬉しいです。

### 🎵 音楽 (Music)
去年の5月に中学時代の友人と映画を観に行った際、その主題歌に深く心を打たれたのがきっかけで音楽を聴くようになりました。
そのアーティストの曲を調べてみると、どれも自分の気持ちに寄り添ってくれるような歌詞ばかりで、今では登下校中に欠かせない楽しみになっています。

### 🧶 編み物 (Knitting)
母が編み物をしていたことに影響を受けて始めました。
最初は失敗ばかりで諦めそうになりましたが、母に教わったり動画で研究したりして練習を重ねるうちに、どんどん楽しくなっていきました。現在はイヤーマフラーを編めるまで上達しています。

### 📺 ドラマ鑑賞 (Drama)
特にミステリー系や医療系のドラマを好んで見ています。
* **主な視聴作品:** 『ナイト・ドクター』『グッド・ドクター』『アンサング・シンデレラ』など
医療ドラマを通じて、人の命の重みや祈りの大切さ、そして困難に立ち向かう医師や看護師の方々の「諦めない気持ち」にとても感銘を受けています。

編み物のように、一つひとつの工程を大切に積み重ねていくことを心がけています。
